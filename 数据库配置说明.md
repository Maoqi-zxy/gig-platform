## 🗄️ 数据库结构

### 数据表

| 表名 | 用途 |
|------|------|
| `users` | 用户表（企业 + 自由职业者） |
| `tasks` | 任务表 |
| `applications` | 申请表 |
| `submissions` | 提交表 |

### 用户表结构 (users)

```sql
CREATE TABLE users (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  username VARCHAR(50) UNIQUE NOT NULL,
  email VARCHAR(100) UNIQUE NOT NULL,
  password_hash VARCHAR(255) NOT NULL,
  role VARCHAR(20) DEFAULT 'freelancer',
  company_name VARCHAR(100),
  skills TEXT,
  avatar_url VARCHAR(255),
  phone VARCHAR(20),
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
)
```

## 🔍 查看本地数据库数据

### 方法 1: 使用 SQLite 命令行

```bash
# 进入数据库
sqlite3 /Users/yu/.homiclaw/workspace-agent-pqh0jg/gig_platform.db

# 查看所有用户
SELECT id, username, email, role, company_name, created_at FROM users;

# 查看所有任务
SELECT id, title, enterprise_id, status, created_at FROM tasks;

# 查看表结构
.schema users

# 退出
.exit
```

## 📋 常用 SQL 查询

### 查看所有用户
```sql
SELECT * FROM users;
```

### 查看企业用户
```sql
SELECT * FROM users WHERE role = 'enterprise';
```

### 查看自由职业者
```sql
SELECT * FROM users WHERE role = 'freelancer';
```

### 查看最新注册用户
```sql
SELECT * FROM users ORDER BY created_at DESC LIMIT 10;
```

### 查看所有任务
```sql
SELECT t.id, t.title, t.description, t.status, u.username as enterprise_name
FROM tasks t
JOIN users u ON t.enterprise_id = u.id;
```
